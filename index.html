Runget — удобная, красивая и быстрая операционная система

Одна система для дома, офиса, серверов, игр и экспериментов.

Runget основан на Debian и Linux-ядре и сочетает стабильность Debian-семейства с расширенной совместимостью с Windows, Linux и Android-приложениями, включая подсистему Andryemu для Android. Система оптимизирована для работы на широком спектре оборудования — от старых ПК с минимальными характеристиками до современных игровых станций и серверов.

Почему именно Runget?

Современный интерфейс в стиле KDE, запуск Windows и Android-приложений, поддержка приложений разных Linux-дистрибутивов и высокая производительность даже на старом железе. Runget объединяет лучшее из трёх миров — Linux, Windows и Android — в одном дистрибутиве.

На базе Debian

Runget использует стабильную платформу Debian, что даёт предсказуемые обновления, безопасность и совместимость с огромным количеством Debian-совместимых пакетов и репозиториев.

Debian — один из старейших и наиболее проверенных Linux-дистрибутивов, лежащий в основе Ubuntu, Linux Mint и сотен других систем. Runget наследует эту стабильную базу и добавляет собственные репозитории.

Debian APT Stable dpkg systemd

Поддержка Windows-приложений

Запускайте Windows-программы и игры через совместимый слой (Wine/Proton и другие решения), чтобы использовать привычный софт без переустановки системы.

Поддерживаются офисные пакеты (Microsoft Office, 1С), графические редакторы (Adobe Photoshop, CorelDRAW), мессенджеры и тысячи других Windows-приложений через Wine и его производные.

Android-приложения

Andryemu — Android-эмулятор и подсистема для Runget, позволяющая запускать Android-приложения, а также работать аналогично на других Linux-дистрибутивах.

Устанавливайте APK-файлы напрямую, используйте мобильные приложения и игры прямо на рабочем столе. Andryemu обеспечивает аппаратное ускорение графики и поддержку сенсорных экранов.

Приложения других дистрибутивов

Runget нацелен на запуск приложений, созданных для Debian, Ubuntu, Linux Mint, Fedora, Arch и других дистрибутивов, за счёт поддержки разных упаковочных форматов.

Универсальные форматы Flatpak и AppImage работают «из коробки», а Snap-пакеты можно установить дополнительно. Это делает Runget одним из самых совместимых дистрибутивов на рынке.

Высокая производительность

Оптимизация под слабые и старые ПК делает Runget актуальным для восстановления старого железа, а также для использования в роли лёгкой основной системы.

Минимальное потребление оперативной памяти (от 512 МБ), быстрая загрузка и отзывчивый интерфейс даже на процессорах 10-летней давности. Runget Core потребляет ещё меньше ресурсов.

Безопасность и приватность

Runget не содержит телеметрии и не собирает данные пользователей. Регулярные обновления безопасности наследуются от Debian, а встроенный файрвол защищает систему из коробки.

Поддержка шифрования диска (LUKS), SELinux/AppArmor, автоматические обновления безопасности и минимальная поверхность атаки благодаря консервативному подходу Debian к выбору пакетов.

Плюсы Runget и Debian-based систем

Runget наследует сильные стороны Debian-семейства и добавляет свои фишки: акцент на совместимость и восстановление других Debian/Ubuntu-систем. Ниже — подробный обзор преимуществ, которые вы получаете, выбирая Runget.

Стабильность и безопасность

Debian-based дистрибутивы известны стабильностью и строгим подходом к обновлениям, что делает базу Runget надёжной для рабочих станций и серверов.

Каждый пакет проходит многоступенчатое тестирование перед попаданием в стабильную ветку. Обновления безопасности выходят оперативно — обычно в течение 48 часов после обнаружения уязвимости.

Огромный выбор пакетов

APT и сторонние форматы (Flatpak, AppImage) обеспечивают доступ к тысячам приложений, а многие инструкции из интернета подходят к Runget по аналогии с Debian и Ubuntu.

Репозитории Debian содержат более 59 000 пакетов. Вместе с Flatpak и AppImage доступно практически любое приложение для Linux.

Простые обновления

Обновление системы и ПО происходит через знакомые команды apt или графические утилиты, что упрощает администрирование и сопровождение.

Одна команда sudo apt update && sudo apt upgrade обновляет всю систему и все установленные программы.

Runget для восстановления систем

Runget может использоваться как инструмент для восстановления Debian/Ubuntu-систем, например, для починки загрузчика или системных файлов.

Загрузившись с USB-флешки с Runget, можно восстановить GRUB, исправить fstab, сбросить пароль root, проверить и восстановить файловые системы других Linux-установок.

Сообщество и документация

За Debian стоит большое сообщество и документация, а для Runget постепенно формируется собственная база гайдов, форумов и чатов.

Большинство инструкций для Debian и Ubuntu применимы к Runget напрямую. Это миллионы статей, видео-уроков и форумных обсуждений по всему интернету.

Удобство для новичков

Debian-семейство считается удобным для перехода с Windows, а Runget добавляет дружественный интерфейс и подсказки, чтобы пользователю было проще адаптироваться.

Привычное расположение панели задач, меню «Пуск», системный трей — пользователи Windows чувствуют себя как дома. Поддержка Telegram-чата для получения помощи от сообщества.

Универсальная совместимость

Runget — один из немногих дистрибутивов, позволяющих запускать приложения из трёх экосистем одновременно: Linux, Windows и Android. Это делает его универсальным решением для любых задач.

Работайте в LibreOffice, запускайте 1С через Wine, общайтесь в мобильном Telegram через Andryemu — всё на одном рабочем столе без перезагрузок.

Лёгкость и скорость

Runget потребляет значительно меньше ресурсов, чем Windows 10/11 или macOS, при этом предоставляя полноценный рабочий стол с богатым функционалом и красивым интерфейсом.

Там, где Windows 11 требует 4GB RAM и 64GB диска, Runget комфортно работает с 1GB RAM и 10GB на диске. Идеально для возрождения старых компьютеров.

Runget на любом устройстве

Универсальная система для компьютеров, ноутбуков, серверов и экспериментальных платформ. Runget адаптируется под возможности вашего оборудования и предлагает оптимальную конфигурацию для каждого типа устройств.

Компьютеры

Полноценная замена Windows на домашнем или рабочем ПК, с поддержкой игр, офисных программ и мультимедиа. Поддержка NVIDIA, AMD и Intel GPU.

Intel/AMD x86_64 • 1+ ГБ RAM • 10+ ГБ диск

Ноутбуки

Оптимизация питания и драйверов помогает сохранить баланс между автономностью и производительностью. Поддержка гибернации и спящего режима.

TLP для батареи • Wi-Fi/BT • Тачпад • Fn-клавиши

Серверы

Runget Core ориентирован на стабильность и минимализм, что подходит для серверных сценариев и виртуализации. Docker и KVM из коробки.

SSH • Docker • KVM/QEMU • Мониторинг • Headless

Телефоны

Экспериментальная поддержка смартфонов и планшетов для энтузиастов и тестирования новых идей. Адаптивный интерфейс для сенсорных экранов.

Экспериментально • Сенсорный UI • Портирование

Raspberry Pi

Планируется специальная сборка для одноплатных компьютеров и IoT-проектов. Поддержка GPIO, камеры и дисплеев. (В разработке)

ARM64 • Broadcom • GPIO • microSD/USB

Orange Pi

Сборки для Orange Pi 3/4/5 с поддержкой Rockchip и Allwinner SoC. Высокая производительность и NPU для ML-задач. (В разработке)

ARM64 • Rockchip • Allwinner • NVMe • NPU

Виртуальные машины

Runget отлично работает в QEMU/KVM, VirtualBox и VMware. Идеально для тестирования, разработки и знакомства с системой перед установкой.

QEMU • VirtualBox • VMware • Hyper-V
